# KYRON UI/UX SYSTEM

## 1. Designziel

KYRON soll wirken wie ein professionelles Sportdaten-Betriebssystem.

Nicht wie:

- Wettseite
- Casino
- Fan-Spielzeug
- Tabellenfriedhof

## 2. Look

- Dark Premium Sports Intelligence
- Deep Navy
- klare Cards
- große Prozentwerte
- saubere Tabellen
- ruhige Charts
- starke Statusbadges
- hochwertiges Admin Center

## 3. Theme Engine

KYRON nutzt Design Tokens.

Keine hart codierten Farben.

Tokens:

- color.background.primary
- color.surface.card
- color.text.primary
- color.accent.primary
- color.status.success
- color.status.warning
- color.status.danger
- color.prediction.favorite
- color.stream.men
- color.stream.women
- chart.palette.primary
- spacing
- radius
- shadow
- typography

Themes:

- Kyron Midnight
- Kyron Dark
- Kyron Light
- Kyron Arena
- Kyron Analyst
- Kyron Scout
- Kyron High Contrast
